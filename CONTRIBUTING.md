Contributor Guidelines and Developer Notes
==========================================

Installing Docker/ Development Containers
-----------------------------------------

**On Ubuntu/GNU Linux,** [follow these instructions.](https://docs.docker.com/engine/install/ubuntu/#install-using-the-repository)

Then:

```sh
sudo apt-get install docker-compose-plugin
```

Add your user to the `docker` group:

```sh
sudo apt-get install docker-compose-plugin
```
